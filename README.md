# Trusted Ideas HR

A full-stack HR management platform built with React and Node.js.

## Features
- Employee management with department structures
- Real-time attendance tracking
- Leave request and approval workflow
- One-click payroll processing
- Performance review cycles
- Analytics dashboard with PDF export
- Multi-step tenant onboarding wizard

## Tech Stack
**Frontend:** React 18, Vite, custom CSS design system  
**Backend:** Node.js, Express, PostgreSQL, Redis, BullMQ  
**DevOps:** Docker, Nginx, GitHub Actions CI/CD  

## Running locally
```bash
npm install
npm run dev
```

Open http://localhost:5173
